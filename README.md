A relational database system designed to manage the complex operations of Last Resort Hotels, a rapidly expanding luxury hotel chain. The system models and tracks rooms (sleeping rooms, suites, and meeting rooms), reservations, guests, hosts, events, facility usage, billing, and staff activity across multiple buildings, floors, and wings.
Built as part of a database systems course, the project includes an Entity Relationship Diagram (ERD), a normalized database implementation (3NF), multi-table analytical queries, and a web interface that surfaces meaningful operational insights — such as room availability, guest billing summaries, and event scheduling — to support hotel staff in coordinating resources and serving guests effectively.
